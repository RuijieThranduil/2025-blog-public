# LLM · Pretraining 预训练

## 知识点：数据处理与清洗
### 定义
- 对原始语料进行过滤、去重、质量控制。
### 核心理解
- 数据质量决定模型上限。
### 关键细节
- 流程：采集 → 去重 → 质量过滤 → 安全过滤。
- 关注语言分布、领域平衡、重复率。
### 例子 / 直觉
- 训练前先“洗数据”，避免模型学噪声。
### 相关知识点
- `预训练范式` `评估方法`

## 知识点：预训练范式
### 定义
- 在无标注语料上进行自监督学习。
### 核心理解
- 目标是学通用能力，不是单任务最优。
### 关键细节
- 常见目标：Causal LM、Masked LM。
- 训练配方需配合模型规模与数据规模。
### 例子 / 直觉
- 先打“通识底座”，再做场景化微调。
### 相关知识点
- `语言模型基础` `SFT`

## 知识点：Scaling Law
### 定义
- 描述性能与参数量、数据量、算力预算的关系。
### 核心理解
- 关键是预算下的最优配比，而不是盲目增大。
### 关键细节
- 三元平衡：Model / Data / Compute。
- 用于训练预算规划和路线决策。
### 例子 / 直觉
- 只增参数不增数据，效果不一定提升。
### 相关知识点
- `数据处理与清洗` `推理优化`

# LLM · Foundations 基础理论

## 知识点：语言模型基础
### 定义
- 语言模型学习序列概率分布，目标是预测下一个 token。
### 核心理解
- 本质：拟合自然语言分布。
- 重要性：是 GPT 类模型的训练目标基础。
### 关键细节
- 链式分解：\(P(x)=\prod_t P(x_t|x_{<t})\)
- 常用指标：PPL（困惑度）。
### 例子 / 直觉
- 给定“今天天气很”，模型预测“好/冷/热”的概率。
### 相关知识点
- `预训练范式` `Transformer 架构`

## 知识点：词向量与 Embedding
### 定义
- 将离散 token 映射到连续向量空间。
### 核心理解
- 本质：把符号表示成可计算的几何对象。
### 关键细节
- 向量距离可表达语义相近性。
- 常与位置编码一起使用。
### 例子 / 直觉
- “猫”和“狗”的向量通常更接近。
### 相关知识点
- `Attention 机制` `RAG 基础`

## 知识点：Attention 机制
### 定义
- 通过 Q/K 相似度对 V 进行加权聚合。
### 核心理解
- 本质：动态信息路由。
### 关键细节
- 公式：\(\text{softmax}(QK^T/\sqrt{d_k})V\)
- 多头注意力捕获多种关系。
### 例子 / 直觉
- 代词会关注其指代实体。
### 相关知识点
- `Transformer 架构` `KV Cache`

## 知识点：Transformer 架构
### 定义
- 以自注意力为核心的序列建模架构。
### 核心理解
- 本质：并行建模上下文依赖。
### 关键细节
- 组件：MHA + FFN + Residual + LayerNorm。
- LLM 主流为 Decoder-only。
### 例子 / 直觉
- 多个“阅读头”并行关注不同关系。
### 相关知识点
- `语言模型基础` `推理优化`

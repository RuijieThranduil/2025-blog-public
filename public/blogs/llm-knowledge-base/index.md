- 对齐路径：Finetuning → Alignment → Evaluation

## 使用方式

- 每个模块都按“知识点卡片”组织。
- 每个知识点都采用固定结构：定义 / 核心理解 / 关键细节 / 例子 / 相关知识点。
- 后续可在每个模块中继续细分更小卡片（公式卡、工程卡、易错点卡）。
